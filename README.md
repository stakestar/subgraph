# StakeStar Subgraph

## Test

```
yarn test
```

## Copy deps

```
yarn cli copy-addresses -n goerli
```

## Prepare goerli config

```
yarn cli prepare -n goerli -t subgraph.template.yaml
```

## Deploy hosted

```
yarn deploy:hosted
```
